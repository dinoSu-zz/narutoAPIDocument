You must replace meowmeowmeow with your personal API key.
Remember — a happy kitten is an authenticated kitten!
You must replace meowmeowmeow with your personal API key.

HTTP Request

GET http://example.com/api/kittens

Parameter Type Required Description
include_cats false false If set to true, the result will also include cats.
available true false If set to false, the result will include kittens that have already been adopted.

Code format

              
[
  {
    "id": 1,
    "name": "Fluffums",
    "breed": "calico",
    "fluffiness": 6,
    "cuteness": 7
  }
]

Page(draft)

列出所有的頁面


GET /api/xxx/pages

Parameter Type Required Description
name Number true
type String true [PAGE|LINK] 分外部連結和一般頁面
url String true
visible Boolean true [true|false] default:[true]頁面上架和下架
seo object false

List all of the pages:

              
[
   {
      "name":"Home",
      "type":"PAGE", //分外部連結(LINK)和一般頁面(PAGE)
      "url":null,

         "title":"",
         "description":"",
         "keyword":"",
         "link_url":""
      }
   },
   {
      "name":"About us",
      "type":"LINK",
      "url":null,
      "visible":true
   }
   //more........
]

Brand(draft)

設定部份的api


GET /api/xxx/brand

Parameter Type Required Description
brand_id Number true this field is same as the table in gundan
name String true Name of the conpany
adderss String true Address of the conpany
phone String true phone of the conpany
logo_media_url String true
favicon_media_url String true
website_default_home_url String true
create_at String true
update_at String true
dirName String true
website_status String true [ OPEN | CLOSED ]
domainName Number true Serial Number
customerType String true 未來可能會有試用客戶
themeBaseId String true 選用主題的id
tracking_services String false 目前會有GA，未來可能會又OG tag

The information of the client:

              
{
    "brand_id":  user_id, //this field is same as the table in gundan
    "name": null, // the name of company
    "adderss": null, // the address of company
    "phone": null, // the phone of company
    "logo_media_url": null, // the url of logo
    "favicon_media_url": null, // the url of favicon
    "website_default_home_url": null,
    "create_at": "",
    "update_at": "",
    "dirName": null,
    "website_status": "CLOSED",//OPEN/CLOSED
    "domainName": "dinodino",
    "customerType": null,
    "themeBaseId": "wu3dmlcmek",
    "tracking_services":[],

}
